# PictPDF 📸 ➡️ 📄

**PictPDF** is a modern, high-performance mobile application built with **Ionic**, **Angular**, and **Capacitor**. It allows users to seamlessly convert their photos and images into high-quality PDF documents directly on their device.

## ✨ Features

- **Multi-Image Support**: Select multiple images from your gallery.
- **Fast Conversion**: Optimized image processing for quick PDF generation.
- **Offline First**: All processing happens on-device for maximum privacy and speed.
- **History Tracking**: Keep track of your previously generated PDFs.
- **Modern UI**: Sleek, premium dark-themed interface designed for the best user experience.

## 🚀 Tech Stack

- **Framework**: [Ionic Framework](https://ionicframework.com/)
- **Logic**: [Angular](https://angular.io/)
- **Native Bridge**: [Capacitor](https://capacitorjs.com/)
- **Styling**: Vanilla CSS / SCSS with a custom design system.

## 🛠️ Installation & Setup

1. **Clone the repository**:
   ```bash
   git clone https://github.com/ardhaa10/PictPDF.git
   cd PictPDF
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run in development mode**:
   ```bash
   ionic serve
   ```

4. **Build for Android**:
   ```bash
   ionic build
   ```

## 📱 Platforms

- **Android**: Fully supported with Capacitor.
- **Web**: Progressive Web App (PWA) ready.
- **iOS**: Compatible (requires macOS for building).
